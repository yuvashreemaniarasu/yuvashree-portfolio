# Portfolio - Next.js

This is a modern portfolio website built with Next.js, migrated from a vanilla HTML/CSS project.

## Features
- **Next.js 15 (App Router)**
- **TypeScript**
- **Vanilla CSS styling** (preserving the original design)
- **Responsive design**
- **Dark/Light Mode**
- **ScrollReveal Animations**
- **EmailJS Integration**

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
